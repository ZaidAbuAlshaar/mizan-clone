# 🧭 دليل ميزان MIZAN — جولة صفحة بصفحة (Walkthrough)

> آخر تحديث: 2026-06-10 · النسخة القوية (Sentinel-2 ‏20م + GLDAS عبر Earth Engine + بث حي)
> التشغيل: backend على `:8000` + واجهة على `:3000` (انظر README «التشغيل السريع»)

---

## 1) `/` — غرفة عمليات ميزان (الشاشة الرئيسية)

**الوظيفة:** نظرة القيادة الوطنية — أين تُسرق المياه الآن؟

| العنصر | ماذا يفعل |
|---|---|
| **الخريطة الكبرى** | MapLibre فوق صور قمر VIIRS حقيقية (NASA GIBS). تقريب ≥ز8 يحمّل طبقة **HLS Landsat 30م** الأحدّ تلقائياً |
| **🛰/◑ مبدّل الخلفية** | قمر صناعي NASA ↔ خريطة داكنة (Carto) |
| **🔴 قمر مباشر (آخر 48س)** | يستبدل الخلفية بآخر تمريرة VIIRS متاحة (قبل يومين ضماناً) — *الستالايت لايف فعلياً* |
| **الطبقات** | حقول مشبوهة (🔴🟠) · **خريطة حرارة الاشتباه** (موزونة بدرجة P4) · أحواض الخطر · مواقع الإنفاذ الحقيقية |
| **نبض الإنذار** | الحقول 🔴 تنبض على الخريطة عند الزوم البعيد |
| **▶ شاهد الزحف الأخضر** | أنيميشن سينمائي: الحقول تظهر سنة بسنة (2016→2026) بسنة أول رصد الحقيقية + عدّاد حي |
| **بطاقة الأزرق العائمة** | 215% استغلال (MWI 2009 عبر IWMI) — تنقر → صفحة الحوض |
| **شريط المؤشرات السفلي** | 6 بلاطات: 🔴103 · 🟠38 · م³ مستردّ · مطر الصيف 2.8مم (NASA حقيقي) · العجز −205 MCM · معدل الضخّ 150% |
| **التنبيهات (يمين)** | أعلى 5 حقول درجةً — تنقر → الطابور مع تركيز |

**لقطة الديمو:** فعّل الهيت ماب + اضغط «شاهد الزحف الأخضر» — أقوى 20 ثانية افتتاحية.

---

## 2) `/basin/azraq` — تفاصيل حوض الأزرق

**الوظيفة:** القصة العلمية الكاملة لحوض واحد — البرهان الثلاثي.

| العنصر | ماذا يفعل |
|---|---|
| **منحنى GRACE** (حقيقي · JPL) | 254 شهراً (2002→2026-04) سماكة الماء المكافئ، فجوة GRACE→GRACE-FO معروضة بشفافية، تنبؤ بـ backtest MAE ±1.01سم. *إشارة إقليمية ~300كم — لا تُنسب لحوض مفرد* |
| **💧 مخزون GLDAS الجوفي** (حقيقي · **Google Earth Engine**) | **توكيد مستقل ثانٍ**: نموذج CLSM مدمج بجاذبية GRACE — 278 شهراً، اتجاه −1.06مم/سنة. من مشروع `mizan-earth-engine` |
| **قياسات آبار الوزارة** (توضيحيموسوم) | −20م (2000–2017) ~1م/سنة → نافذة العتبة الحرجة 2031–2035 — *العتبة على الآبار لا على GRACE* |
| **خريطة الحوض** | **خلفية القمر تتبع سنة آلة الزمن** — اسحب السحّاب وشاهد الصحراء تخضرّ على صور القمر نفسها؛ سنة 2026 = 🔴 بث حي |
| **⏳ آلة الزمن (بانل)** | سحّاب 2016↔2026: عدد الحقول الظاهرة + الهكتارات + م³/سنة لكل سنة |
| **الميزان المائي NASA POWER** | (مطر − تبخر مرجعي) تراكمي منذ 2002 — عجز هابط من بيانات حقيقية |
| **شاهد النفي المطري** | 2.8مم متوسط مطر الصيف (23 سنة): أي خضرة صيفية = ضخّ جوفي حتماً |
| **دفتر الميزان** | كفّتا تدقيق الكتلة: فقد GRACE الإقليمي ↔ ما نفسّره → «العجز المجهول» (بنيته منتج، أرقامه demo موسومة) |

**لقطة الديمو:** اسحب سحّاب السنين ببطء 2016→2026 — الخلفية والحقول يتغيران معاً، وعند 2026 يظهر شعار البث الحي.

## 3) `/timemachine` — آلة الزمن السينمائية

**الوظيفة:** الدليل البصري الصارخ — «كيف كانت الصحراء وكيف صارت».

| العنصر | ماذا يفعل |
|---|---|
| **مسرح قبل/بعد** | صور **Sentinel-2 حقيقية بدقة 20م** (كانت MODIS 250م — أوضح ×12) · «قبل» = Landsat 8 صيف 2016 · فجوات الأرشيف تُملأ تلقائياً من Landsat |
| **سحّاب الانقسام الذهبي** | اسحب لمقارنة 2016 ↔ السنة المختارة على نفس الإطار تماماً |
| **▶ تشغيل** | autoplay سينمائي: يمرّ على 2018→2025 تلقائياً مع crossfade |
| **شريط السنوات** | 2018 · 2020 · 2022 · 2024 · **2025** |
| **لوحة التحليل (يسار)** | كشوف الآبار حتى السنة المختارة + الهكتارات + معدل التغذية + مصفوفة الخطر |
| **📈 نمو الزحف الأخضر** | مخطط تراكمي (هكتارات ذهبي + حقول تركوازي) 2016→2026 من سنوات أول رصد الحقيقية |

**لقطة الديمو:** اضغط ▶ واصمت — الصور تتكلم.

---

## 4) `/queue` — طابور التفتيش (المنتج التشغيلي)

**الوظيفة:** تحويل العلم إلى عمل ميداني — ماذا يفعل المفتش غداً صباحاً؟

| العنصر | ماذا يفعل |
|---|---|
| **الطابور المرتّب** | 141 حقلاً بترتيب درجة الاشتباه (89 فأدنى) + فلاتر (الحالة/العلم/الدرجة) |
| **بانل الدليل لكل حقل** | • **صورة قبل/بعد حقيقية** للحقل نفسه (Landsat 2016 ↔ Sentinel-2 ‏10م صيف 2025) بشارة «حقيقي · Sentinel-2»<br>• تفكيك الدرجة الشفاف (35+25+15+12.5+12.5 — عقد §3)<br>• سلسلة NDVI الشهرية × المطر (بصمة الريّ anti-phase)<br>• الإحداثيات للـ GPS |
| **دورة الحالة** | جديد → تمّ التفتيش → مؤكَّد/بريء — تُحفظ في SQLite عبر الـ API (انتقال خاطئ = 422) |

**لقطة الديمو:** افتح AZQ-0001 (درجة 89) — صورة قبل/بعد لحقله بالضبط + «لماذا هذه الدرجة» رقماً رقماً.

---

## 5) `/impact` — عدّاد الأثر

**الوظيفة:** ترجمة الكشف إلى أرقام قرار: م³/سنة قابلة للاسترداد بنطاق (لا رقم واحد)، سيناريوهات معدل تأكيد قابلة للتحريك (سحّاب 10–90%)، وما يعنيه ذلك من حصص مياه.

---

## 6) `/validation` — التحقّق P7

**الوظيفة:** الصدق العلمي — مواقع إنفاذ حقيقية منشورة تُقارن بمضلعاتنا (عتبة ≤5كم معلنة سلفاً) + precision@20. النقاط الذهبية/التركوازية على الخريطة الرئيسية عند تفعيل طبقة «مواقع الإنفاذ».

---

## 7) `/methodology` — المنهجية والحدود

**الوظيفة:** درع أسئلة الحكّام: CORE/SUPPORT/STRETCH، صيغة الدرجة (model card)، الحدود الخمسة المعلنة، إفصاح الذكاء الاصطناعي، ولماذا «القاعدة قبل الـ ML».

## 🛰 سلسلة المصادر (كلها حقيقية)

| الطبقة | المصدر | الوصول |
|---|---|---|
| كشف الحقول 141 | Sentinel-2 L2A NDVI | Planetary Computer (بلا مفتاح) |
| TWS + تنبؤ | GRACE/GRACE-FO mascon JPL RL06.3Mv04 | Earthdata token |
| **GWS الجوفي** | **GLDAS-2.2 CLSM** | **Google Earth Engine (`mizan-earth-engine`)** |
| المناخ + النفي المطري | NASA POWER (276 شهراً) | بلا مفتاح |
| الخلفية + البث الحي | VIIRS/MODIS GIBS + HLS L30 | بلا مفتاح |
| آلة الزمن 20م | Sentinel-2 + Landsat 8 | بلا مفتاح |

**الاستثناء الوحيد الموسوم «توضيحي»:** عتبة منسوب الآبار (استقراء من أرقام MWI منشورة) + أرقام دفتر الميزان (بنيته هي المنتج) + سلسلة sm_rootzone في خصائص الحقول (مولّدة موسمياً، ليست SMAP).
