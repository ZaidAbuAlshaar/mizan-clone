## 2) انضباط الأرقام — أين ومتى

- **المرجع الوحيد:** `plan.md` → **ملحق أ** (أ.1 المشكلة + أ.2 المال). عند تضارب: الملحق يحسم والرقم المخالف يُحذف فوراً.
- الأرقام المرساة: 61 م³/فرد · 205 م.م³ ضخّ جائر · 215% الأزرق / 176% عمّان-الزرقا (MWI **2009** عبر IWMI — لا "2017") · 201 بئر/62 م.م³ (2023/24) · 1593 بئراً منذ 2013 · 6 مليارات $/300 م.م³ الناقل · 0.5–0.7$/م³ تحلية · 1 م.م³ ≈ 3,730 شخصاً · الأزرق −20م (2000–2017) · ~1م/سنة وطنياً.

### صياغات الأمان (إلزامية في كل موضع ظهور)

| الرقم | الصياغة الآمنة الوحيدة |
|---|---|
| **308 آلاف م³** | متوسط **البئر المُغلَق في حملات الإغلاق 2023/24** — عيّنة منحازة للكبار. **لا يُضرب × 1500 أبداً** |
| **70%** | من **عدد** الآبار العاملة زراعية — الحجم ~42% من السحب. لا نخلط |
| **~1500 بئر** | **تقدير 2014** — العدد الحقيقي مجهول (نقطة بيع: هذا ما يحلّه ميزان) |
| **القيمة المالية** | تقديرية بمنهجية معلنة (م³ × تحلية 0.5–0.7$) — تُعلن في الشاشة/الشريحة نفسها |

### قاعدة صياغة GRACE (لكل عنوان شاشة وشريحة)

- السلسلة الزمنية = **«منحنى GRACE للمنطقة الشرقية/الأردن»** (إشارة إقليمية ~300كم — الشارة إلزامية بجانب المنحنى).
- هبوط الحوض = **«قياسات آبار الوزارة (−20م)»** · تحديد الحقول = **«Sentinel-2»** حصراً.
- لا ادعاء حقلي من الجاذبية أبداً. MASCON يُقطع عند 9/2024 بشفافية + فجوة 2017–2018 معروضة.
- العتبة الحرجة تُعرَّف على **مناسيب الآبار** (نطاق 2031–2035) لا على TWS anomaly.

## 4) أوامر التشغيل

```bash
# 1) توليد البيانات التجريبية (حتمي — يكتب إلى data/demo/)
python tools/generate_demo_data.py

# 2) الـ backend (FastAPI على :8000 — راجع backend/README لتفاصيل الوحدة)
cd backend && pip install -r requirements.txt && uvicorn main:app --reload --port 8000

# 3) الواجهة (Next.js على :3000)
cd web && npm install && npm run dev
# نسخ بيانات الـ fallback (الواجهة لا تسقط عند غياب الـ API):
#   data/demo/*  →  web/public/data/
```

- الواجهة تقرأ بنمط fallback: `NEXT_PUBLIC_API_URL` أولاً ثم `web/public/data/*.json` — **الديمو لا يسقط أبداً**.
- دورة حالة الحقل: `new → inspected → confirmed | cleared` (أي انتقال آخر = 422).

## 6) مصائد شائعة (من المراجعة — لا تكرّرها)

- معرّف GLDAS الصحيح: `NASA/GLDAS/V022/CLSM/G025/DA1D` (باند `GWS_tavg`) — **لا** `GRACE_DA`.
- MODIS ET: `MOD16A2GF` للفترة 2018–2020 + `MOD16A2` من 2021 (فخ التغطية الزمنية).
- WorldCover: فلتر موسَّع `{cropland, bare, tree cover, shrubland}` (الزيتون) أو قناع استبعاد فقط.
- محمية الأزرق الرطبة (RAMSAR) + الواحة **مستبعدتان** من P2 (`data/demo/exclusions.geojson`).
- صيغة درجة P4 المعتمدة **بلا طبقة تراخيص**: `35·inside + 25·new + 15·persistence/12 + 12.5·norm(area) + 12.5·norm(expansion)` — 🔴≥70 · 🟠40–69 · 🟢<40.
- تحقّق P7: عتبة **≤5كم معلنة سلفاً** + lift (null model) + صياغة «تطابق حالات» — مواقع وادي الأردن خارج المنهجية (out_of_methodology) بشفافية.
